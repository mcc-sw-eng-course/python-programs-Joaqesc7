To run the files and obtain the statemente coverage, the following steps have to be taken into consideration:

Install the coverage tool with the command 'pip install coverage'
Run test cases using the following command 'coverage run HW2_TEST.py'
Then run the following command to obtain the report 'coverage report -m'
